# SudokuSolver
Simple Sudoku Solver using DFS 

### How to use


    #### To Build
    in a linux environment run the make command
    ```
    make
    ```

    This will create a GroupSudokuSolver.x that can then be run to solve
    all boards in the ./boards/ folder

    #### To Run

    ```
    ./GroupSudokuSolver.x
    ```
